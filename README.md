### 규칙
매주 월요일 22시~23시 사이 디스코드를 통해 각자 공부한 것에 대한 마크다운 문서를 보며 얘기하고 다음 주 주제를 정한다.

일요일까지 올리고 일요일 날 각자 문서 읽고 난 후 월요일 날 이야기하도록 함.

책 읽고 문서 작성 한 뒤에 마지막에 책을 읽으면서 궁금했던 점 or 궁금한 점 써놓고 이야기 할 수 있도록 한다.
